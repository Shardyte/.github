# Shardyte
Shardyte is an early-stage project exploring a **cross-game value and item system** designed to make progression, rewards, and ownership portable across different game servers and worlds.
The core idea is simple: allow players to earn value in one game environment and use it in another, while keeping **game servers in control** of how that value is created, traded, and applied in gameplay.
This organization exists to host ongoing development, experiments, and documentation as the project evolves.


## Why Shardyte Is Being Developed?
Many games and servers keep progression and value locked to a single world, which limits what players can carry forward and makes it difficult for server owners to build systems with lasting impact. Shardyte is being developed to explore whether value, ownership, and progression can exist beyond one server while still respecting each gameâ€™s rules and design. The idea is to give players a way to invest time and effort into things that can hold value across communities, while giving server owners new tools to reward engagement, experiment with economies, and add meaning to items, achievements, and participation. Without forcing a shared economy or taking control away from the servers themselves.

## Direction
Shardyte is being designed with the following principles in mind:
- Game-first design, not blockchain-first
- Server-authoritative integrations
- Optional participation for games and communities
- Low-cost, scalable infrastructure
- Support for tokenized in-game items
The goal is to build shared systems that different games can adopt without being forced into a single economy or ruleset.
